The **Employee Task Management System** is a web-based enterprise-style application developed using **Oracle APEX and PL/SQL**.

The application demonstrates **real-world database-driven application development**, combining SQL, PL/SQL, validations, and APEX UI components to deliver a scalable and maintainable solution.

This project is built to reflect **industry-level standards** and is suitable for showcasing practical skills in **Oracle APEX development, database design, and PL/SQL programming**.
